HMAC-SHA256: сид = ключ. Сообщение = svc + len + счётчик ct. Rejection Sampling: max = 256 - (256 % длина алфавита), байты >= max отбрасываются. | HMAC-SHA256 (логин): ключ = сид, сообщение = "login" + svc. Логин детерминирован: одинаковый сид + сервис дают одинаковый логин.
