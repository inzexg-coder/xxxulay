# passgen

Детерминированный генератор паролей на основе HMAC-SHA256.

## Установка и запуск

### Linux (Arch, Ubuntu и др.)

```bash
git clone git@github.com:inzexg-coder/xxxulay.git
cd xxxulay
python3 app.py
```

### Windows

```bash
git clone git@github.com:inzexg-coder/xxxulay.git
cd xxxulay
python app.py
```

> Программа сама откроется в отдельном окне браузера без адресной строки.
> Если не установлен Chromium-браузер, откроется в обычном браузере.

### Зависимости

Не требует установки дополнительных пакетов. Всё работает на стандартной библиотеке Python и встроенном Web Crypto API браузера.
