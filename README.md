# passgen

Детерминированный генератор паролей на основе HMAC-SHA256.

**passgen** — это программа, которая создаёт криптостойкие пароли без необходимости их хранить. Достаточно запомнить одну секретную фразу (сид) и название сервиса — пароль всегда можно восстановить заново. В основе лежит алгоритм HMAC-SHA256: пароль вычисляется детерминированно, то есть одинаковые входные данные всегда дают одинаковый результат. Это позволяет обходиться без менеджеров паролей, облачных хранилищ и бумажных записей.

Программа предоставляет гибкие настройки генерации: регулируемая длина, выбор наборов символов (заглавные/строчные/цифры/спецсимволы), опции инверсии (первая заглавная, последняя цифра), готовые пресеты, а также генерацию логина. Для каждого сервиса автоматически запоминаются индивидуальные настройки. Встроенный индикатор надежности оценивает качество пароля по длине и разнообразию символов.

Интерфейс написан на HTML/CSS/JavaScript с использованием Web Crypto API, серверная часть — на Python (стандартная библиотека, без внешних зависимостей). Программа работает на Linux и Windows, открывается в отдельном окне браузера без адресной строки. Экспорт и импорт настроек в JSON позволяют переносить конфигурацию между устройствами. Все вычисления происходят локально — никакие данные не передаются по сети.

## Установка и запуск

### Linux (Arch, Ubuntu и др.)

```bash
git clone git@github.com:inzexg-coder/xxxulay.git
cd xxxulay
python3 app.py
```

### Windows

```bash
git clone git@github.com:inzexg-coder/xxxulay.git
cd xxxulay
python app.py
```

> Программа сама откроется в отдельном окне браузера без адресной строки.
> Если не установлен Chromium-браузер, откроется в обычном браузере.

### Зависимости

Не требует установки дополнительных пакетов. Всё работает на стандартной библиотеке Python и встроенном Web Crypto API браузера.
