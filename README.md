# passgen

**passgen** — детерминированный генератор паролей на основе HMAC-SHA256. Достаточно запомнить секретную фразу (сид) и название сервиса — пароль всегда можно восстановить заново. Никаких менеджеров паролей, облачных хранилищ и записей.

Все вычисления происходят локально в браузере через Web Crypto API. Данные не передаются по сети и не хранятся между сессиями.

## Возможности

- Генерация криптостойких паролей через HMAC-SHA256
- Регулируемая длина (1–99 символов)
- Выбор наборов символов: заглавные, строчные, цифры, спецсимволы
- Инверсия: первая заглавная + последняя цифра
- Готовые пресеты (минимальный, стандартный, максимальный)
- Визуальный эмодзи-код (4 иконки) для верификации сида
- Индикатор надёжности пароля
- Сохранение настроек по сервисам (в сессии)
- Экспорт/импорт настроек в JSON с выбором сервисов
- Встроенный список сервисов: google, github, vk, yandex, mail, icloud, onliner, kufar
- Добавление пользовательских сервисов

## Запуск

Откройте `index.html` в браузере:

```
# Windows
double-click index.html

# Linux
xdg-open index.html
# или
python3 app.py
```

## Требования

- Современный браузер (Chrome 90+, Firefox 90+, Safari 15+, Edge 90+)
- Поддержка Web Crypto API
- Работает на Windows, Linux, macOS
