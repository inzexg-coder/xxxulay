# passgen

Детерминированный генератор паролей на основе HMAC-SHA256.

## Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone git@github.com:inzexg-coder/pract.git
cd pract
```

### 2. Запустить программу

**Linux (Arch, Ubuntu и др.):**

```bash
python3 app.py
```

**Windows:**

```bash
python app.py
```

> Программа сама откроется в отдельном окне браузера без адресной строки.  
> Если не установлен Chromium-браузер, откроется в обычном браузере.

### Зависимости

Не требует установки дополнительных пакетов. Всё работает на стандартной библиотеке Python.
